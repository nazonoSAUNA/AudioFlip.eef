# 反転(音声) - 拡張編集フィルタプラグイン
- フィルタオブジェクトと音声のフィルタ効果に追加されます

- [ダウンロードはこちら](../../releases/)
- AudioFlip.eefをpluginsフォルダに配置してください

- このプラグインを読み込むためにはpatch.aul r43_ss_58以降が必要です https://scrapbox.io/nazosauna/patch.aul

## パラメータ
- 左右（チェック）：ステレオ音声の場合、左右の音を入れ替えます
- 位相（チェック）：音の波の向きを反転させます
